# Thursday demo script — one paragraph

> "Good morning. What you're seeing is the **CEFANET Digital Notice Board** — a civic-technology platform that puts every kwacha of the Constituency Development Fund in public view. Nationally we're tracking **K6.24 billion across 158 constituencies**; today I'm going to walk you through five live constituencies — Lusaka Central, Mandevu, Kabulonga, Kabwe Central, and Livingstone. On the dashboard you can immediately see total projects, allocated budget, expenditure, average completion, and active alerts; the chart underneath compares budget against actual spend per category. Switching constituencies refreshes everything live — this is real data, queried from a Postgres database via Drizzle ORM, with a typed Express API and Zod-validated contracts shared end-to-end with the Next.js frontend. The Project Registry gives you every project with status, budget, and completion; clicking any row opens a detail drawer with GPS coordinates, the contractor, and a full update history timeline. The Financial Overview shows utilisation, variance, and a six-month spending trend. The Map plots every project geolocated and colour-coded by status — green for complete, blue for ongoing, red for stalled, grey for planned. The Bursary Tracker — tagged SDG-4 — anonymises beneficiaries while still showing the gender split and the distribution across primary, secondary, and tertiary education. And critically, the Alert Engine: any project that stalls automatically surfaces here with days-overdue and budget-at-risk — right now you can see **three stalled projects per constituency** flagged, totalling tens of millions of kwacha. The whole platform is fully responsive — every screen works on a 375-pixel mobile device with a bottom navigation bar. The login supports three demo accounts: a CEFANET super admin with full access, a district officer scoped to Lusaka Central, and the unauthenticated public view you're looking at now. This entire system runs locally on a single machine with one `npm install`, one Docker command, and one seed — production migration to AWS Cognito, managed Postgres, and Redis caching is a configuration change, not a rewrite. **K6.24 billion deserves a notice board. This is it.**"
